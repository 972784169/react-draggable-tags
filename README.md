# react-draggable-tags
An easy and flexible tag component

## Documentation
See: https://ygyooo.github.io/react-draggable-tags

## Installation
```sh
npm install --save react-draggable-tags
```

```js
import {DraggableArea} from 'react-draggable-tags';
```
